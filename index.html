  • a set of animated objects
    • The two differences here is that the update inside the object will change a value (an x or something like that). Every time the update is called the value changes.
    In the main js file there is a new function called animate. This now runs in a loop in real time. It runs about 40 times per second.
    Every time it loops it erases the entire canvas and re-draws all the objects (though their update).
    Because the a variable ( like x ) has changed, the object appears in a different position and we have the illusion of movement because the changes happen very quickly.
    The counters should give you an idea of how quickly the animation loop is running.
